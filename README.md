# signal

Signal is a lightweight prototype for a family investments dashboard.

## What is in this repo
- `dashboard/` — current comparison shell for the dashboard approaches
- `signal v.1.0/` — original portfolio-oriented dashboard prototype
- `signal v.1.1/` — alternative dashboard focused on cash, location, and FX visibility

## Current state
The current top-level dashboard acts as a comparison shell between the earlier approaches.

## Notes
This repository was extracted from a larger local workspace and cleaned up into a standalone project.
